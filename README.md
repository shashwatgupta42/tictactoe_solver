# Tic-Tac-Toe Solver
This project uses minimax (https://en.wikipedia.org/wiki/Minimax) to play tic-tac-toe against the user. 

      - bot.py - Tic Tac Toe bot with simple terminal interface
      - bot_gui.py - Tic Tac Toe bot with GUI (built using pygame)
      - demo - folder containing a demo video
